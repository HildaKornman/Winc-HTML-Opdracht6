Welkom bij HTML Opdracht 6 Uitgebreid HTML Formulier

Deze opdracht is om te oefenen met een uitgebreid HTML Formulier op de Recepten webpagina. De vorige opdracht was om een basic formulier te laten zien op de contactpagina. Eerder is de styling veranderd als het scherm breder wordt dan 1024 pixels. Echt leuk om te zien hoe dat er uitziet en als inleiding van mijn eerste unordered lijst:

  • De eenvoud
  • De compacte tekst
  • De zinloze inhoud met lelijke opmaak
Dit is nog een andere paragraaf over hoe leuk ik deze opdracht vind. Wel jammer dat het niet zichtbaar is hoe handig deze pagina georganiseerd is nu met allerlei HTML layout elements. Ik kan meteen uitproberen of de live server goed werkt. Dit is heel erg handig; het werkt! De CSS styling is nog wel een uitdaging.
